Bienvenue sur Webserv

Webserv est un projet de serveur web développé en C++ visant à implémenter les fonctionnalités essentielles d'un serveur web moderne. Ce projet inclut la gestion des requêtes HTTP, le support des scripts CGI, le téléversement et la suppression de fichiers, ainsi que la gestion des fichiers statiques. Notre objectif est de fournir une solution performante, sécurisée et facile à utiliser pour héberger vos applications web.
